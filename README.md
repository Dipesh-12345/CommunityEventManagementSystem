# Community-Event-Management-System
A Community Event Management System built using ASP.NET Core Blazor

# Project Overview
The Community Event Management System is a web-based application designed to manage community events, participants, venues, activities, and approval workflows.

# Features
# Participant
- View events
- View event details
- Submit registration requests
- View registration status
- Request withdrawal

# Administrator
- Create and update events
- Manage venues
- Manage activities
- View pending registration requests
- Approve or decline participant requests

# Technologies Used
- ASP.NET Core
- Blazor
- Entity Framework Core
- SQL Server
- Bootstrap
- C#

# Security Features
- Role-based access control
- Input validation
- Secure authentication for admin users

# Testing
- Manual test cases
- CRUD operation testing
- Approval workflow testing

#  How to Run the Project
1. Clone the repository
2. Open the solution in Visual Studio
3. Update the database connection string
4. Run the application
